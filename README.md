# QuickPaste
Global hotkey manager for quick text insertion

## Download
[QuickPaste.exe](./QuickPaste.exe) - ready-to-use Windows version

## How to use
1. Download exe file
2. Run - app goes to system tray
3. Add texts with hotkey combinations
4. Use Alt+Z, Ctrl+Shift+1 and other shortcuts
5. Multiple entries show selection menu
